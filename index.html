Minhas Habilidades

As tecnologias abaixo representam as principais ferramentas que utilizo para transformar ideias em aplicações web modernas, acessíveis e com boa experiência para o usuário.

HTML

Linguagem utilizada para estruturar e organizar o conteúdo das páginas web.

CSS

Responsável pela aparência visual, cores, espaçamentos e layout das páginas.

JavaScript

Linguagem que adiciona interatividade e dinamismo às aplicações web.

TypeScript

Extensão do JavaScript que adiciona tipagem e ajuda a desenvolver aplicações mais seguras.

React

Biblioteca utilizada para criar interfaces modernas e reutilizáveis para aplicações web.

Python

Linguagem versátil utilizada em automação, análise de dados, inteligência artificial e desenvolvimento de software.

PHP

Linguagem amplamente utilizada para desenvolvimento de aplicações e sistemas web no servidor.

SQL

Linguagem utilizada para consultar, armazenar e gerenciar informações em bancos de dados.

Node

Ambiente que permite executar JavaScript no servidor e construir APIs e aplicações Back-End.
